# 🧠 Concept: Memory Corruption_Stack Buffer Overflow

## 📌 Definition
Buffer: 데이터가 목적지로 이동되기 전에 보관되는 임시 저장소
Stack Buffer: Stack에 있는 지역 변수
Heap Buffer: Heap에 할당된 메모리 영역
Buffer Overflow: Buffer가 넘치는 것

### 📄 Important Data Tampering
Buffer Overflow가 발생하는 Buffer 뒤에 중요한 데이터가 있다면 해당 데이터가 변조 됨으로써 문제 발생 가능
