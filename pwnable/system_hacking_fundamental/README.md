# 🛡️ pwnable_writeups

Personal research and practice repository for binary exploitation, memory corruption, and bypassing security mitigations.

## 🎯 Goal
- Master memory corruption vulnerabilities (Stack/Heap Overflow, Type Confusion, etc.)
- Bypass modern exploit mitigations (NX, ASLR, Canary, PIE, RELRO)
- Craft reliable and sophisticated exploit payloads using pwntools

## 🛠 Environment
- Ubuntu (VM & Local) & Windows (WSL)
- GDB (pwndbg)
- checksec / ROPgadget / objdump
- Python3 (pwntools)
- VS Code

## 📚 Learning Platform
- Dreamhack

## 🧠 Study Method
- Analyze binary protections (`checksec`) and identify vulnerability location
- Understand stack/heap layout and memory execution flow
- Find a way to bypass mitigations by leaking memory addresses
- Document the exploitation flow, payload structure, and root cause
