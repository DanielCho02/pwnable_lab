# 🌐 web_hacking_writeups

Personal research and practice repository for web application security, vulnerability analysis, and bug bounty hunting.

## 🎯 Goal
- Master client-side and server-side vulnerabilities (XSS, SQLi, SSRF, CSRF, etc.)
- Understand web architecture, HTTP/S protocols, and authentication mechanisms
- Develop exploit chaining skills and secure coding awareness

## 🛠 Environment
- Chrome DevTools / Firefox Developer Edition

## 📚 Learning Platform
- Dreamhack


## 🧠 Study Method
- Map the attack surface and understand the application logic
- Identify input vectors and potential vulnerability classes
- Craft precise payloads to bypass filters or WAFs
- Document the exploitation flow, impact, and mitigation strategies
